How this skill supports Horizon Mining Systems

Mining technology companies rely on connected systems, user accounts, operational data and remote access. Cyber security helps reduce the risk of unauthorised access, data loss and downtime.

My studies have introduced me to security concepts such as authentication, secure configuration, incident response, vulnerability awareness and the importance of protecting sensitive information.

Example work

I can document risks, explain basic controls, and test systems in a safe and authorised learning environment. For a mining company, this could include checking that internal web forms do not expose unnecessary data and that users are trained to avoid social engineering.

Workplace value

This skill is valuable because mining operations depend on availability and trust. Strong security habits help keep equipment, staff and business information protected.
